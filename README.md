# Server setup

## Setup 
- npm install typeorm -g
- npm install

## Run the app:

### Run app:
```
    npm run start # Sirve para correr en produccion
    npm run start:dev # Sirve para correr un servidor en desarrollo
    npm run start:debug # Sirve para correr el manejo de depuracion
``` 

### Run build of app:
```
    npm run build
```
